# Python Design System

## Init project

Virtualenv

```bash
python -m venv .venv
./.venv/Scripts/activate
```

Pylint

```bash
pip install pylint
```

Pre Commit

```bash
pip install pre-commit
pre-commit install
```

Database ORM

```bash
pip install SQLAlchemy
```

Testes

```bash
pip install -U pytest
```
